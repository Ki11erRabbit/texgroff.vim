# texgroff.vim
A plugin to open groff/latex output pdf in preview

### Requirements

* **groff** with the `groff_ms` macro
* **texlive-full** for pdflatex
* **zathura** pdf viewer

### Keybindings

| Leader | Key | Action |
|:------:|:---:|:------:|
| 	/		 |  p  | Open PDF Preview |
| 	/		 |  q  | Compile groff(ms)/latex(tex) |
